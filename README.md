CPFMove package is designed for facilitating delineation and filtering of foraging trips directly from raw GPS data. It offers basic functionality to filter data and tools for evaluating the sensitivity of parameter selection. The package is particularly intended for species displaying a central-place foraging behavioral pattern

Package can be downloaded using:

```r
library(devtools)
install_github("PhilBertrand/CPFMove")
```

Documentation with basic information on how to use the
CPFMove functions is available at [CPFMove Introduction](https://philbertrand.github.io./)
